## Consolidación de compras por estilo

Luego del análisis, se crea un archivo 'results.csv', el cual contiene el resultado final.


# Como utilizar el programa:

1. Cargar documentos en la carpeta 'data' con formato '.xlsx'.

2. Abrir y ejecutar el archivo 'main.py'.

3. Revisar resultados en la carpeta 'results'.
